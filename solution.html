Solutions

Lynis

Lynis est un outil fiable pour la sécurité informatique qui fonctionne sur les plateforme Linux, macOS ou d'autres systèmes d'exploitation basés sur Unix. Cet outil scanne en détails un système informatique à la recherche de vulnérabilité afin d'améliorer sa sécurité. Ce projet est open source avec la licence GPL et est accessible depuis 2007.

Installation

Lynis peut être utilisé sur Mac ou Linux. Ce programme permet d’effectuer des audits de système. Il teste si l’ordinateur peut empêcher qu’un logiciel malveillant ou autre rentre dans le système de l’ordinateur. Il teste également la détection de vulnérabilités et la sécurité du système. Installation Pour installer lynis sur votre ordinateur vous avez deux méthodes. La première est d’installer à l’aide de github avec le lien suivant : Link .
Cliquer sur Code en vert qui va installer le code nécessaire sur votre ordinateur. Sur le terminal de votre ordinateur, créer un clone de ce code en tapant : git clone https://github.com/CISOfy/lynis. Puis exécuter ce code en tapant sur le terminal, cd lynis; ./lynis audit system. Vous pouvez également exécuter le code en tant que root. Sinon sur votre machine Linux, en prenant en compte que votre système est soit un système Debian, soit Linux Mint, soit Ubuntu, soit un autre système se basant sur ces derniers, taper sur votre terminal : apt-get install lynis. Pour l’exécuter, tapez ./lynis. Pour effectuer des audits système, tapez : lynis audit system.

ClamAV

ClamAV est un antivirusGPL pour UNIX. La principale qualité de cet antivirus est qu'il permet de balayer les courriels reçus et envoyés avec un logiciel de messagerie classique. Le paquet que nous allons installer inclut un démon multi-tâches flexible et configurable, un antivirus en ligne de commande et un utilitaire pour une mise à jour automatique des définitions de virus via Internet. Le programme est basé sur une bibliothèque distribuée avec le paquet ClamAV, que vous pouvez utiliser pour créer votre propre logiciel. Le plus important est que la base de données des virus soit mise à jour.

Installation

Vous pouvez télécharger via le lien suivant : Lien
Pour un système d’exploitation Linux de type Ubuntu, tapez dans le terminal : apt-get install clamav.
Le lien précédent comporte des indications en plus pour un système d’exploitation Linux tels que Ubuntu ou encore Debian dans la catégorie Linux Packages. De même, pour les Mac vers le bas de la page.

Nessus

Nessus détecte les machines vivantes sur un réseau, balaie les ports ouverts, identifie les services actifs, leurs versions, puis tente diverses attaques. Cet outil permet un audit de sécurité en s'appuyant sur deux éléments : un client et un serveur. Le serveur, nessusd, est chargé de tester le système indiqué en essayant toutes les attaques que sa base contient, pendant que le client, Nessus (qui n'est qu'une interface graphique), fait un rapport sur les différents résultats obtenus.

Installation

Accédez à la page de téléchargements Nessus et récupérer le package d'installation d'Ubuntu. Nessus 8.13.1 est la version stable actuelle au moment de la rédaction de cet article. Vous pouvez simplement exécuter la commande ci-dessous pour télécharger le programme d'installation de Nessus 8.13.1 Ubuntu :
curl -o Nessus-8.13.1-Ubuntu1110_amd64.deb
https://www.tenable.com/downloads/api/v1/public/pages/nessus/downloads/12207/download?i_agree_to_tenable_license_agreement=true&file_path=Nessus-8.13.1 -Ubuntu1110_amd64.deb
Ensuite, installez la version d'essai professionnelle de Nessus sur Ubuntu 20.04 en exécutant la commande ci-dessous :
apt install ./Nessus-8.13.1-Ubuntu1110_amd64.deb
Démarrez et activez le service Nessus Professional pour qu'il s'exécute au démarrage du système.
systemctl enable --now nessusd

OWASP ZAP

OWASP ZAP est un outil Java permettant de tester la sécurité des applications Web. Il possède une interface graphique intuitive et de puissantes fonctionnalités permettant de fuzzing, de scripting, de spidering, de proxy et d'attaque d'applications Web. Il est également extensible à travers un certain nombre de plugins. De cette manière, il s'agit d'un outil de test d'applications Web tout-en-un. OWASP ZAP pourrait même devenir votre outil de test d'applications Web une fois que vous aurez compris le principe.

Installation

Accédez à la page de téléchargements OWASP ZAP et récupérer le package d'installation d'Ubuntu. Ensuite exécuter la commande suivante : sudo ./ZAP_2_10_0_unix.sh
La configuration de OWASP ZAP se fait en interface graphique, ainsi veuillez suivre les instructions pour poursuivre l’installation de OWASP ZAP.
Après installation, lancer OWASP ZAP.

Snort

Snort est un système de détection d'intrusion (ou NIDS) libre publié sous licence GNU GPL. Il est capable d'effectuer aussi en temps réel des analyses de trafic et de logger les paquets sur un réseau IP. Il peut effectuer des analyses de protocole, recherche/correspondance de contenu et peut être utilisé pour détecter une grande variété d'attaques et de sondes comme des dépassements de buffers, scans, attaques sur des CGI, sondes SMB, essai d'OS fingerprintings et bien plus.

Installation

A l'origine, ce fut un sniffer qui connu une telle évolution qu'il fut vite adopté et utiliser dans le monde de la détection d'intrusion en s'appuyant sur une base de signature régulièrement enrichie par le "monde du libre". Sous Linux (comme sous Windows) son installation est simple et se résume (pour linux) par les commandes suivantes, une fois l'archive téléchargée dans le répertoire "/usr/local/snort". Ainsi, on effectue le lien entre snort et mysql afin d'utiliser une base de donnée pour la détection d'intrusion. L'outil sera alors bien plus riche et réactif. Suricata

Suricata est un logiciel open source de détection d'intrusion, de prévention d'intrusion, et de supervision de sécurité réseau. Il permet l'inspection des Paquets en Profondeur (DPI). De nombreux cas d'utilisations déontologiques peuvent être mis en place permettant notamment la remontée d'informations qualitatives et quantitatives.

Installation

L’installation de Suricata se fait en 3 étapes :
Etape 1 : Créer Atlantic.Net Cloud Server
Tout d'abord, connectez-vous à votre Atlantic.Net Cloud Server  . Créez un nouveau serveur , en choisissant Ubuntu 20.04 comme système d'exploitation, avec au moins 2 Go de RAM. Connectez-vous à votre serveur Cloud via SSH et connectez-vous à l'aide des informations d'identification mises en évidence en haut de la page. Une fois que vous êtes connecté à votre serveur Ubuntu 20.04, exécutez la commande suivante pour mettre à jour votre système de base avec les derniers packages disponibles : apt-get update -y
Etape 2 :  Installer les dépendances requises

Tout d'abord, vous devrez installer certaines dépendances nécessaires pour compiler Suricata à partir de la source.  Vous pouvez tous les installer avec la commande suivante :
apt-get install rustc cargo make libpcre3 libpcre3-dbg libpcre3-dev build-essential autoconf automake libtool libpcap-dev libnet1-dev libyaml-0-2 libyaml-dev zlib1g zlib1g-dev libcap-ng-dev libcap-ng0 make libmagic-dev libjansson-dev libjansson4 pkg-config -y
Par défaut, Suricata fonctionne comme un système de détection d'intrusion (IDS). Si vous souhaitez inclure la fonctionnalité du système de prévention des intrusions (IPS), vous devrez installer d'autres packages sur votre système.  Vous pouvez les installer avec la commande suivante :
apt-get install libnetfilter-queue-dev libnetfilter-queue1 libnfnetlink-dev libnfnetlink0 -y
Une fois tous les packages installés, vous devrez installer l'outil de mise à jour suricata pour mettre à jour les règles Suricata.  Vous pouvez l'installer avec les commandes suivantes :
apt-get installer python3-pip
pip3 install --upgrade suricata-update
ln -s /usr/local/bin/suricata-update /usr/bin/suricata-update
Etape 3 : Installer Suricata
Tout d'abord, téléchargez la dernière version de Suricata depuis leur site officiel avec la commande suivante :
wget https :// www . openinfosecfoundation . org / téléchargement / suricata - 5.0.3 . goudron . gz
Une fois le téléchargement terminé, extrayez le fichier téléchargé avec la commande suivante :
tar -xvzf suricata-5.0.3.tar.gz
Ensuite, remplacez le répertoire par le répertoire extrait et configurez-le avec la commande suivante : cd suricata-5.0.3
./configure --enable-nfqueue --prefix=/usr --sysconfdir=/etc --localstatedir=/var
Ensuite, installez le Suricata avec la commande suivante :
make
make install-full

Legion

Legion est un outil graphique basé sur python open-source et facile à utiliser qui peut aider dans les tests de pénétration du réseau. Legion est une version améliorée de Sparta. Auparavant, Sparta était préinstallé avec Kali Linux, maintenant Kali a supprimé Sparta en raison de son ancienne version Python (Python2). Legion est très similaire à Sparte. Ce cadre de test de pénétration de réseau semi-automatisé est installé avec Kali Linux.

Installation

Pour installer les composants dockers généralement nécessaires et ajouter la configuration de l'environnement pour docker, sous un terme, on exécuteles commandes suivantes:Sudo apt-get legion
sudo apt-get update
sudo apt-get install -y docker.io python-pip -y
sudo groupadd docker
pip install --user docker-compose
Pour installer les composants dockers généralement nécessaires et ajouter la configuration de l'environnement pour la redirection docker, sous la fenêtre WSL, exécutez :
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release cs) stable"
sudo apt-get update
sudo apt-get install -y docker-ce python-pip -y
sudo apt autoremove
sudo usermod -aG docker $USER
pip install --user docker-compose
echo "export DOCKER_HOST=tcp://localhost:2375" >> ~/.bashrc && source ~/.bashrc
